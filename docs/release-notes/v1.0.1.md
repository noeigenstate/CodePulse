## v1.0.1

### 更新内容

- 修复 Codex 额度条偶发停留在 Spark：按当前活跃或最近使用的模型显示周额度，切换到 gpt-5.6 等普通模型后不再误显示 Spark。
- 缩小安装包体积：纯 JS 依赖打入主进程、排除预览图与可选软 Vulkan 组件（保留 ffmpeg，Electron 启动必需）。
- 完善 macOS 首次打开说明：未签名包出现 “is damaged” 时用 `xattr -cr` 去除隔离标记（「仍要打开」通常无效）。
- 修复 macOS 从 Finder 启动时检测不到 CLI：自动探测 Homebrew / nvm 等常见路径（不仅依赖 GUI 的最小 PATH）。
