## v0.1.10

### 更新内容

#### 多 CLI 仪表盘

- 新增 Grok 第三分屏，支持 Codex / Claude Code / Grok 三端并排展示。
- 仪表盘改为自适应分屏：仅显示当前有任务（或仍保留额度）的 CLI 栏。
- 移除右上角整体状态按钮，界面更简洁。

#### 额度与上下文

- 同步 Grok 上下文窗口占用与当前模型（读取本地 session `signals.json`）。
- 同步 Grok 每周额度（读取本地 billing 日志中的 SuperGrok credit 用量）。
- Codex / Grok 额度条仅显示每周限制（不再展示 5 小时额度）；Claude Code 仍保留 5 小时 + 周额度。
- 修复 Codex 取消 5h 后周额度无法同步：按 `window_minutes` 识别 primary/secondary（周额度常落在 primary）。

#### 应用内更新

- 修复更新过程卡住：下载超时与校验、显示下载进度，启动安装器后立即退出，避免 NSIS/UAC 被挡住。
- 加速自动更新下载：多连接分段拉取、复用已下载完整安装包，优先走加速镜像并在失败时自动切换源。
- 更新下载超时后自动打开 GitHub Release 页，便于手动安装。

#### 文档

- 更新中英文 README，补充 Grok hook 配置与自适应分屏说明。
