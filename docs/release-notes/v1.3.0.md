## v1.3.0

### 更新内容

#### 模型与思考信息更可靠

- 根据本机 CLI 会话中的最新配置稳定展示实际使用的模型，减少同一任务内模型标签在候选值之间跳变的问题。
- 可读取的思考深度会与模型信息一同显示；Claude Code 的思考深度会随本地配置更新或清除。

#### CLI 原生耗时同步

- 后台会从 Codex、Claude Code 与 Grok 的本地会话数据间歇同步任务耗时。
- 进行中的任务持续显示耗时；任务完成后保留最终耗时，刷新或重启后也能从 CLI 本地数据恢复。
- 改善旧会话与过期同步数据处理，避免静态记录错误恢复任务状态或导致耗时不准确。

#### 外观与面板设置

- 右上角新增设置入口，可选择显示哪些 CLI 工具；隐藏面板不会停止本机同步或通知。
- 新安装默认启用自动主题：本机时间 **08:00–20:00** 使用白色主题，其余时间使用黑色主题。
- 设置中可选择自动、白色或黑色主题；已有手动白/黑偏好会继续保留。

#### 同步与交互优化

- 优化本机会话、额度与状态刷新流程，降低无效刷新带来的界面抖动。
- 修正后台刷新按钮的图标显示，并补充模型、耗时、设置与同步场景的自动化测试。
