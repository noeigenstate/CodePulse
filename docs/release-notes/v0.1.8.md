## v0.1.8

### 更新内容

- 修复 v0.1.7 安装包版本不匹配导致的重复更新提示问题。
- 修复 Claude Code 60 秒空闲提醒被误判为“等待输入”的问题。
- 忽略 Claude Code 登录成功等普通通知，避免制造错误的“需要介入”状态。
- 增加 SQLite 旧事件和 token 快照自动清理，降低长期运行后的数据库体积。
- 修复 CodePulse hook 写入 matcher 组、误删用户同名 hook、非原子写配置等配置安全问题。
- 优化托盘更新频率和隐藏任务过滤，减少高频状态刷新带来的干扰。
- 修复 Claude 上下文百分比回退计算，避免用累计输入高估当前上下文。
- 增加异常状态过期移除，避免错误、取消和用量上限状态长期占据面板。
